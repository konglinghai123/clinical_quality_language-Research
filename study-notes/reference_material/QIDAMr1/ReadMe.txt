This package contains materials for the HL7 Health Quality Improvement, Domain Analysis Model Release 1.

In addition to the specification document, the following supplemental files are included

* QDM-vMR-cross-map.xlsx - a file showing mappings among QIDAM, QDM, and VMR specifications
* QIDAMr1.eap - a UML class model of QIDAM. A viewer for Microsoft Windows computers is available at http://www.sparxsystems.com/bin/EALite.exe
* QIDAM-htmldocs.zip - Documentation generated from the UML model. This document shows the attributes for all class and interface attributes on one page.
