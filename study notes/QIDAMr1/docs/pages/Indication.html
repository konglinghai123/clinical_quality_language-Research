action/common

Class Indication

  • An asserted clinical reason justifying an action such as to prescribe a medication, or to perform a procedure or a test. The reason can be specified as a code or as another statement, e.g., code for diabetes (ICD-9-CM 250.0) or Condition (with diabetes code) documented elsewhere in a patient's record.
