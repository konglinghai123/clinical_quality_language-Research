Class ActionStatus

  • Class describing the status of an action.

    • Field Summary

      Fields 
      Field Type and Description
      reason Code
      A coded reason for the status. This is used typically when the status indicates the action was changed, canceled, rejected, or not performed. E.g., patient declined.
      status Code
      A coded value for the status, e.g., Completed, Rejected, Pending.
      statusUpdateTime TimePoint
      The date and time when the status was updated.
